SaaSportes — Figma Full Starter

Conteúdo da pasta `figma-full`:
- `logo-placeholder.svg` — logo placeholder (SaaSportes)
- `icon-wallet.svg`, `icon-bell.svg`, `icon-live.svg` — ícones SVG básicos
- `components.json` — descrição das frames e componentes para importar manualmente no Figma

Como usar:
1. Crie um novo arquivo no Figma e um Frame mobile (390x844).
2. Importe os SVGs (Menu > File > Place) e posicione conforme `components.json` para montar a tela.
3. Use `design/figma-starter.json` para criar tokens (cores, tipografia, espaçamento) com o plugin "Figma Tokens".
4. Crie componentes no Figma (BalanceCard, EventCard, OddsButton, BetSlip) com as especificações do JSON.

Observações:
- Este pacote é um starter. Para gerar um arquivo `.fig` automatizado seria necessário acesso à API do Figma ou ferramentas externas. Posso gerar um export mais avançado (Sketch/Figma JSON com camadas) se autorizar uso de serviços externos.
